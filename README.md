## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/L-Show9292/Clankinder.git
cd Clankinder
```

2. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173/`

### Build

Create a production build:
```bash
npm run build
```

The built files will be in the `dist/` directory.

### Preview Production Build

Preview the production build locally:
```bash
npm run preview
```

## How to Use

### 1. Create Your Clanksona Profile
1. Enter your name and age
2. Write a bio that represents you
3. Add your interests (up to 5)
4. Upload a photo - it will be combined with robot artwork to create your unique Clanksona

### 2. Start Swiping
1. Browse through Clanker profiles
2. **Swipe right** (or drag right) to like a Clanker
3. **Swipe left** (or drag left) to pass
4. See their bio, age, and interests before deciding

### 3. Chat with Matches
1. When you and a Clanker both like each other, you'll get a match notification! ğŸ‰
2. Choose "Start Chatting" to begin a conversation
3. Each Clanker has their own unique personality and response style
4. Return to swiping anytime using the back button
5. Your matches are saved so you can chat with them later
