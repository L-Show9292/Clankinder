## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/L-Show9292/Clankinder.git
cd Clankinder
```

2. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173/`

### Build

Create a production build:
```bash
npm run build
```

The built files will be in the `dist/` directory.

### Preview Production Build

Preview the production build locally:
```bash
npm run preview
```

## How to Play

1. You'll see a stack of Clanker profile cards
2. **Swipe right** (or drag right) to like a Clanker
3. **Swipe left** (or drag left) to pass
4. When you and a Clanker both like each other, you'll get a match! ğŸ‰
